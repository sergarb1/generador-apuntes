# 📚 Plantilla de Apuntes Educativos - CC BY 4.0

Esta carpeta contiene la plantilla base para generar **apuntes educativos profesionales** con licencia abierta.

## 📄 Licencia

Todo el material generado con esta plantilla está bajo licencia **Creative Commons Atribución 4.0 Internacional (CC BY 4.0)**.

### ¿Qué puedes hacer?

✅ **Compartir** — Copiar y redistribuir el material en cualquier medio o formato

✅ **Adaptar** — Remezclar, transformar y crear a partir del material para cualquier propósito, incluso comercialmente

### ¿Bajo qué condición?

⚠️ **Atribución** — Debes dar crédito apropiado, proporcionar un enlace a la licencia, e indicar si se hicieron cambios. Puedes hacerlo de cualquier manera razonable, pero no de una manera que sugiera que el licenciante te respalda o respalda tu uso.

## 🎨 Elementos de la Plantilla

### Banner Superior (Web)
- Muestra el logo CC BY 4.0
- Explicación breve de la licencia
- Enlace al texto completo

### Footer (Web)
- Sección destacada con información completa de la licencia
- Iconos explicativos de permisos
- Atribución al autor/a y tutor/a
- Logo pequeño CC BY

### Sección de Impresión (PDF)
- Se añade automáticamente al imprimir/exportar a PDF
- Incluye logo CC BY 4.0
- Texto de atribución completo
- No es visible en la versión web

```

├── README.md           # Este archivo
├── data/

│   ├── introduccion.js # Contenido de introducción
│   └── ...             # Más archivos de datos

```

## 🛠️ Cómo Usar la Plantilla

### 1. Generación Automática (Recomendado)

Ejecuta el script de inicio y deja que la IA genere los apuntes:

```bash
./Iniciar-Asistente.sh
```

### 2. Manual

1. Copia la carpeta `plantilla/` a tu proyecto
2. Edita `data/metadata.js` con la información del apunte
3. Edita los archivos `data/*.js` con el contenido
4. Abre `index.html` en tu navegador

## 🎓 Uso en el Aula

Esta plantilla está diseñada para:

- **Profesores**: Crear material didáctico abierto para sus clases
- **Alumnos**: Generar apuntes personales para estudio
- **Centros educativos**: Producir recursos institucionales compartibles

## 📝 Ejemplo de Atribución

Cuando uses material generado con esta plantilla, atribuye así:

```
"[Título del Apunte]" por [Nombre del Autor/a], bajo licencia CC BY 4.0
Disponible en: [URL si corresponde]
```

### Ejemplo concreto:

```
"La Célula - Biología 3º ESO" por María García, bajo licencia CC BY 4.0
```

## 🌟 Beneficios de CC BY en Educación

1. **Colaboración**: Los profesores pueden compartir y mejorar materiales
2. **Accesibilidad**: Todo el mundo puede usar los recursos gratuitamente
3. **Adaptación**: Cada profesor puede adaptar el material a sus necesidades
4. **Calidad**: La revisión por pares mejora los recursos
5. **Sostenibilidad**: Reduce la duplicación de esfuerzos
