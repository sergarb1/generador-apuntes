Genera apuntes profesionales para tu aula

Este sistema crea material didáctico profesional para Primaria, ESO, Bachillerato y Formación Profesional. Cada apunte incluye objetivos de aprendizaje, contenido estructurado, ejemplos resueltos, ejercicios graduados y autoevaluación.

⚙️ Cómo funciona

💻 Descarga los agentes para Gemini CLI, Qwen Code o Copilot CLI en el repositorio GitHub. Encuentra instrucciones detalladas de instalación y uso.

1️⃣

Descarga los agentes

Clona el repositorio de GitHub con todos los agentes especializados para tu CLI preferido.

2️⃣

Configura el entorno

Sigue las instrucciones para configurar Gemini CLI, Qwen Code o Copilot CLI.

3️⃣

Solicita el apunte

Indica el nivel educativo (Primaria, ESO, Bachillerato o FP), asignatura, curso y tema.

4️⃣

Agentes especializados

Los agentes de IA analizan y estructuran el material pedagógicamente.

5️⃣

Recibe tu apunte

Obtén un apunte profesional en HTML con objetivos, contenido, ejemplos y ejercicios.

6️⃣

Usa en tu aula

Imprime en PDF, exporta a Word o comparte directamente con tus estudiantes.

🤖 Agentes Especializados

El sistema utiliza múltiples agentes de IA que trabajan de forma coordinada para crear apuntes de alta calidad:

🎯

Agente Orquestador

Coordina todo el proceso y distribuye las tareas entre los agentes especializados.

📖

Agente Experto en Proyectos

Diseña la estructura pedagógica y los objetivos de aprendizaje del tema.

✍️

Agente Generador de Contenido

Redacta el contenido principal con ejemplos, explicaciones y casos prácticos.

✅

Agente de Evaluación

Crea ejercicios graduados, actividades de autoevaluación y rúbricas.

🎨

Agente de Formato

Da formato HTML profesional, responsive y accesible al contenido.

🔍

Agente Revisor

Verifica la calidad pedagógica, ortografía y coherencia del material.
